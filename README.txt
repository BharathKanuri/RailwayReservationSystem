Usage:

Select a route and enter details (separated by space) to book a ticket.
The ticket will be saved in Ticket.txt.
Optionally, cancel the booking to generate a cancellation receipt in CancelTicket.txt.

Concepts Demonstrated:

Class and Object-Oriented Programming
Inline Functions
Inheritance (Multi-level)
File Handling
Exception Handling
Standard Template Library (STL)
